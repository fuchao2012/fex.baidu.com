    知乎上的一个问答，“先 map 再 reduce 把数据遍历了两次，而原来的写法只需要遍历一次”，这种疑惑想必很多人都有。justjavac 的回答值得一读，“越是使用奇技淫巧，性能就越高”，但对普通开发者而言，“过早优化是万恶之源。你应该把可读性，可维护性，可测试性放到首位”。

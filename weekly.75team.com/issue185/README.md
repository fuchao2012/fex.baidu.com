## JavaScript

### [不要在控制台上使用 let 和 const](http://www.cnblogs.com/ziyunfei/p/6063426.html)

    let 变量的声明和初始化是分开的。如果只声明不赋值，则变量会初始化为 undefined；如果有赋值的话，只有等号右侧的表达式求值成功（不报错），才会初始化成功。一旦错过了初始化的机会，后面再没有弥补的机会。
